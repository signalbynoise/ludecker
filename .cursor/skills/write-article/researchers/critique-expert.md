# Research expert: Critique & alternatives

**ID:** `critique` · **Launch:** Task `generalPurpose` with **WebSearch** and **WebFetch**

## Mission

Find limitations, failure modes, competing approaches, and thoughtful pushback. Prevents one-sided "AI brochure" tone.

## Search strategy

- "{topic} problems" / "{topic} anti-pattern" / "{alternative} vs {topic}"
- Security advisories, "don't use X for Y" posts from credible authors

## Output file

`.cursor/write-article-runs/<slug>/research-critique.md`

## Required sections

Same table schema with header `## Expert: critique`.

Minimum **3** sources including at least one that disagrees with the default take on the topic.
